# visual

:package:A project created by vue-cli-3 can be deployed independently or as a sub app in micro-frontends use qiankun

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Deploy to gh-pages

```
yarn run deploy
```
